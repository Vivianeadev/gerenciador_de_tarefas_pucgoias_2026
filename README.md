V9 Gerenciador de Tarefas

Sistema web para gerenciamento de tarefas com quadro Kanban e calendário integrado. Desenvolvido como trabalho para o Projeto Integrador III-B da PUC Goiás.

Funcionalidades

- Login simples (digite seu nome – dados salvos localmente)
- Criação e exclusão de projetos
- Quadro Kanban com colunas: A fazer, Em andamento, Concluído
- Arrastar tarefas entre colunas (drag and drop)
- Adicionar tarefas com título e prazo
- Calendário visual com os prazos das tarefas
- Persistência de dados no navegador (localStorage)

Tecnologias utilizadas

- HTML5
- CSS3 (com design responsivo e tema luxuoso)
- JavaScript puro
- FullCalendar (biblioteca para calendário)
- LocalStorage (armazenamento local)

Como executar

1. Abra com qualquer navegador moderno (Chrome, Firefox, Edge)
2. Digite um nome e clique em "Entrar"
3. Comece a gerenciar seus projetos e tarefas!

Estrutura do projeto

index.html único arquivo contendo todo o código (HTML, CSS, JS)
